Planit: A platform to plan a successful event without hassle

Features:

All:
1. register with email
2. Login (Admin hardcoded)
3. Forgot password? Reset with email
4. register with number
5. logout
6. Dashboard: See all the recent activities
7. Browse
8. chat
   
Vendor:

9. Portfolio; Vendors need portfolio to be able to verify themselves to the admin. Without it, they cannot request for verification
10. Verification: To prove that the vendors are legit, they have to verify themselves to the admin. Without it, they won't get any opportunity to get shown on browsing page or take any order.
11. photography vendor + venue vendor: They can either create packages that a client can take, or list up their facilities with hourly rate for the client to choose
12. catering vendor: These category vendors can also offer package per person or let the client decide on the menu for event. 
13. Decoration + custom photobooth: They will let the client choose which decoration they want. If enabled then they can also let the client make their own custom photobooth
14. collab: Vendors can collab with each other to create offered packages(upto 2 vendors of any category)

Client:

15. Event creating and management: Host can request to book vendors based on their availablility
16. Event timeline+notification
17. Private guestlist + custom invitation sending option
18. Public signup: For public events, the host can create poster and share a link. Public can access the link and sign up for the event without signing in to the website. They only have to use email OTP to confirm
19. Stalls: Public events can have stalls and host can take booking through this website. Booking process is similar to public signup
20. Budget tracker: Dynamically adjusts to show how much is spent and left
21. Checklist: Preexisting + can be customized so you don't forget anything
21. Helpbot/ Ask Away: if the client has any query regarding website.
22. Reviews and ratings: Once the event is closed, the host can write their opinion about the vendor
23. photo sharing: For private event, host can send link to guests which would let them to share the moment from guest's perspective. The host can also download and share the photos by zipping the and sending them over mail.

25. Payment (simulated money): The Host will pay the admin including 5% charge. Admin will cut 5% more as fee from vendor and give the rest back to vendors. The admin can choose to release the money separately or all at once. This would help if the client had any issue with any vendor and want a different solution.
